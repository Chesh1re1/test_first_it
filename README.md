# Управление движением денежных средств (ДДС)

Веб-приложение на Django для составления и анализа таблиц движения денежных средств (ДДС / Cash Flow). Позволяет отслеживать финансовые потоки, управлять доходами и расходами через удобную административную панель с фильтрацией по датам.

## Возможности

- Полный учет финансов — регистрация всех поступлений и списаний
- Категоризация операций — группировка доходов и расходов по статьям
- Фильтр по датам — анализ движения средств за любой период прямо в админ-панели
- Удобное управление — интуитивная админ-панель Django для редактирования записей
- SQLite — легкая база данных без дополнительной настройки

## Технологии

- Python 3.11+
- Django 6.x
- SQLite

## Установка и запуск

### Предварительные требования
- Python 3.11 или выше
- pip

### Пошаговая установка

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/Chesh1re1/test_first_it.git
   cd test_first_it
   ```
2. **Создайте и активируйте виртуальное окружение**
    ```bash
    python -m venv venv
    # Для Windows:
    venv\Scripts\activate
    # Для Linux/Mac:
    source venv/bin/activate
   # Перейтиде в папку проекта
   cd dds
    ```
3. **Установите зависимости**
    ```bash
   pip install -r requirements.txt
    ```
4. **Настройте базу данных**
    ```bash
   python manage.py migrate
    ```
5. **Создайте администратора**
    ```bash
   python manage.py createsuperuser
   ```
6. **Запустите сервер**
    ```bash
   python manage.py runserver
    ```
7. **Откройте приложение**  
Основной сайт: http://127.0.0.1:8000  
Админ-панель: http://127.0.0.1:8000/admin

## Использование  
### Работа с админ-панелью
Войдите в админ-панель под созданной учетной записью  
Добавьте необходимые статусы, типы, категории и подкатегории в соотвествующие справочники  
Добавляйте записи о доходах и расходах  
Используйте фильтр по датам для просмотра операций за нужный период  
Редактируйте или удаляйте записи при необходимости
